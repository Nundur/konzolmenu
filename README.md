# Bemutatkozás - konzolmenu
Ez a program egy saját fejlesztésű konzol ui keretrendszer. Kis mellék projektként indult tanulás miatt, de több minden is kiforrt belőle.
A projekt sajnos nem natívan egyből windows/linux konzolra irat ki, hanem a beépített Console osztály függvényeit használja (Write, WriteLine, stb) különböző komponensek kiiratására, így egy picit lassabb mint más framework alternatívák, de ez a projekt az egyszerűségre épül és a tisztaságra:)

# Komponensek:
- Grid (még fejlesztés alatt)
- GroupBox (`asd`)
- Line
- MTextBlock (WPF : `Textblock`)
- MTextBox (WPF : `Textbox`)
- MenuLista (WPF : `Listbox`)
- ProgressBar (WPF : `Progressbar`)
- TextBlock (WPF : `Textblock`)
- TextBox (WPF : `Listbox`)
